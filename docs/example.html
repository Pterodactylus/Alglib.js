Constrained Minimization Arbitrary Constraints

This is an example of the solution of constrained optimization using ALGLIB.js

Let \(x_1\) and \(x_2\) denote the intputs to a function \(f(x_1,x_2)\) where \[f(x_1,x_2) = 2*|x_0|+|x_1|.\] Subject to the equality: \[x_0=1\] Subject to the inequality: \[x_1>=-1\]
